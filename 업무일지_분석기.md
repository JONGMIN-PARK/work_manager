# 업무일지 분석기 v6

> 주차별 관리 · 월간/연간 트렌드 · 프로젝트 달력 & 타임라인 · AI 인사이트

## 개요

업무일지 데이터를 업로드하여 **주간/월간/연간** 단위로 팀의 업무 투입현황을 분석하는 웹 도구입니다.
IndexedDB 기반 로컬 저장소를 활용하여 주차별 데이터를 누적 관리하고, 복수 주차를 선택해 트렌드를 시각화합니다.
v6에서는 **프로젝트 달력**, **간트 타임라인**, **관리자 대시보드**가 추가되어 프로젝트 일정 관리가 가능합니다.

## 설정 파일 구조

### config.js — 상수/설정

| 상수 | 설명 |
|------|------|
| `TH` | 테마 정의 (7종: auto, light, midnight, forest, sand, rose, slate) |
| `ENC` | 지원 인코딩 목록 |
| `COL` | 팀원 구분 색상 팔레트 (15색) |
| `AM` | 업무분장 코드 → 한글명 매핑 |
| `ABG`, `AFG`, `ABR` | 업무분장 배경/전경/차트 색상 |
| `CMP_COL`, `CMP_DASH`, `CMP_PT` | 인원 비교 차트 스타일 |
| `AI_CONFIG` | AI 제공자별 설정 (모델명, URL, 키 플레이스홀더 등) |
| `DIFF_KEYWORDS` | 난이도 분석용 키워드 + 가중치 |
| `AI_PRESETS` | AI 빠른 지시사항 프리셋 |

### settings.js — 기능 모듈

| 기능 | 함수 | 저장 키 |
|------|------|---------|
| **별칭 관리** | `setAlias()`, `getAlias()`, `displayName()`, `shortName()` | `wa-aliases` |
| **그룹 관리** | `createGroup()`, `updateGroup()`, `deleteGroup()`, `applyGroup()` | `wa-groups` |
| **별칭 UI** | `renderAliasModal()`, `applyAliasModal()`, `clearAllAliases()` | — |
| **그룹 UI** | `renderGroupModal()`, `renderGroupList()`, `renderGroupQuickButtons()` | — |
| **LS 래퍼** | `lsGet()`, `lsSet()`, `lsDel()` | — |

### order.js — 수주번호 ↔ 수주명 매핑

```javascript
var ORDER_MAP = {
  "S2024-001": "○○설비 제작",
  "D2025-010": "△△시스템 개발",
  ...
};
```

- **로드**: 페이지 시작 시 ORDER_MAP을 기본값으로 로드
- **우선순위**: localStorage(`wa-oc-*`) > ORDER_MAP(order.js)
- **저장**: 화면에서 수주명 저장 시 localStorage + order.js 파일 동시 저장
- **파일 저장**: File System Access API(Chrome) 지원 시 직접 저장, 미지원 시 다운로드
- **외부 편집**: 텍스트 편집기에서 직접 수정 후 새로고침하면 반영

## 주요 기능

### 1. 주간 분석 (Weekly)

- **파일 업로드**: `.xls`(HTML/바이너리), `.xlsx` 지원, 드래그앤드롭 또는 클릭
- **인코딩 전환**: `euc-kr`, `utf-8`, `cp949`, `shift_jis`, `iso-8859-1`
- **팀원 선택**: 단일/다중 선택, 즐겨찾기, 즐겨찾기 필터
- **팀원 그룹**: 다중 선택 → 이름 지정하여 그룹 저장, 빠른 적용 버튼
- **팀원 별칭**: 원래 이름에 대한 별칭(닉네임) 지정, 전체 UI에 반영
- **상세 필터**: 수주번호, 업무분장(A~S), 업무내용 키워드 검색
- **수주번호 상세**: 선택 수주 관련 인원 표시, 수주명/코멘트 관리
- **통계 카드**: 필터 결과 건수, 총 투입시간, 기간, 일평균
- **상세 테이블**: TSV 복사, 엑셀 저장, 인라인 편집 모드
- **차트**: 업무분장 비율(파이), 인원별 분포(바), 일별 투입시간(바)
- **인원 비교**: 2명 이상 선택 시 비교 분석
- **AI 요약**: Gemini / Claude API 연동, 커스텀 지시사항 + 프리셋
- **내용 요약**: 선택 인원 전체 업무 내용 요약 (난이도, 분산도 분석 포함)

### 2. 아카이브 (Archive)

- **주차 저장**: 주차 라벨 + 데이터 + 별칭 스냅샷을 IndexedDB에 저장
- **원본 파일 포함**: 원본 바이너리 보관 → 인코딩 변경 후 재파싱
- **주차 목록**: 카드 형태 표시, 별칭 포함 팀원명 표시
- **상세 보기**: 분장 분포, 인원별 요약, AI 리포트

### 3. 트렌드 분석 (Trend)

- **복수 주차 선택** → 업무분장별/인원별 추이 차트
- **AI 월간 인사이트**: 트렌드 데이터 기반 AI 분석

## 데이터 구조

### 입력 파일 컬럼

| # | 컬럼명 | 타입 | 설명 |
|---|--------|------|------|
| 0 | 날짜 | `YYYYMMDD` | 엑셀 시리얼/`YYYY-MM-DD` 자동 변환 |
| 1 | 이름 | 문자열 | 팀원명 |
| 2 | 수주번호 | 문자열 | 프로젝트/수주 식별자 |
| 3 | 시간 | 숫자 | 투입시간 (소수점 허용) |
| 4 | 업무분장 | 문자열 | 업무 유형 코드 |
| 5 | 약자 | 문자열 | 업무분장 약자 (A/B/D/G/M/S) |
| 6 | 업무내용 | 문자열 | 상세 업무 내용 |

### 업무분장 코드 (config.js → AM)

| 약자 | 의미 |
|------|------|
| A | CS현장 |
| B | 제작 |
| D | 개발 |
| G | 일반 |
| M | 관리 |
| S | 영업지원 |

### 저장소

| 저장소 | 키/스토어 | 내용 |
|--------|-----------|------|
| IndexedDB | `WorkAnalyzerDB` → `weeks` | 주차별 데이터, 원본 파일, 별칭 스냅샷 |
| localStorage | `wa-theme` | 현재 테마 |
| localStorage | `wa-fav-names` | 즐겨찾기 팀원 목록 |
| localStorage | `wa-fav-only` | 즐겨찾기 필터 토글 |
| localStorage | `wa-aliases` | 팀원 별칭 맵 |
| localStorage | `wa-groups` | 팀원 그룹 목록 |
| localStorage | `wa-oc-{수주번호}` | 수주명 코멘트 (order.js 오버라이드) |
| 파일 | `order.js` → `ORDER_MAP` | 수주번호↔수주명 기본 매핑 |
| localStorage | `wa-ai-prov` | AI 제공자 선택 |
| localStorage | `wa-ak-{provider}` | AI API 키 |
